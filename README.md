## Installation

There are some instruction to install this project on your system

### Clone the repository
```
git clone https://github.com/skmirajulislam/QuizGame.git
```
### Navigate to the project directory
```
cd QuizGame
```
### Install dependencies
```
npm install
```
### Install dependencies
```
npm run dev
```
